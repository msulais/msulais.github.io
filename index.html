About Me Tentang Saya

Hi, I'm Muhammad Sulais, a Front-End Web Developer with a passion for UI/UX design. With 3 years of experience, I've honed my skills in HTML, CSS, and TypeScript/JavaScript. I specialize in frameworks like React, Next.js, SolidJS, Svelte + SvelteKit, and Astro, and I'm proficient in using Figma for design. Hai, saya Muhammad Sulais, seorang Pengembang Web Front-End yang memiliki minat pada desain UI/UX. Dengan pengalaman 3 tahun, saya telah mengasah keterampilan saya dalam HTML, CSS, dan TypeScript/JavaScript. Saya ahli dalam framework/library seperti React, Next.js, SolidJS, Svelte + SvelteKit, dan Astro, dan saya ahli dalam menggunakan Figma untuk desain.

My journey in Front-End development Perjalanku kepada Front-End development

My journey in Front-End development started in high school. When COVID-19 hit in 2020, I found myself with ample free time at home. Searching for something interesting to do, I stumbled upon Python. My initial thought was to build desktop applications, so I created a stopwatch timer app using TKinter. However, I soon realized that Python was too slow for application development, so I switched to C++ for its speed. Perjalanan saya di dunia Front-End development dimulai saat masih di SMA. Ketika COVID-19 melanda di tahun 2020, saya punya banyak waktu luang di rumah. Untuk mencari kegiatan menarik, saya mulai mengenal Python. Awalnya, saya mencoba membuat aplikasi desktop dengan TKinter, seperti stopwatch timer. Namun, saya cepat menyadari bahwa Python terlalu lambat untuk pengembangan aplikasi, sehingga saya beralih ke C++ karena kecepatannya.

Unfortunately, my progress with C++ stalled as I couldn't find an ideal GUI for building desktop apps. Everything was challenging to set up, even the Win32 API. My laptop also wasn't powerful enough to run Visual Studio smoothly, further complicating matters. Sayangnya, kemajuan saya dengan C++ terhambat karena sulit menemukan GUI yang ideal untuk membuat aplikasi desktop. Semua opsi yang ada, termasuk Win32 API, terlalu rumit untuk diatur. Selain itu, laptop saya tidak cukup kuat untuk menjalankan Visual Studio dengan lancar, yang menambah tingkat kesulitan.

In January 2021, I gave up on desktop app development due to the lack of a straightforward advertising platform. Instead, I turned to mobile app development and began learning Flutter to build Android apps. I developed around 10 apps and uploaded them to the Google Play Store. However, in October 2024 I had to hide them all due to Google's new policy requiring my personal address to be displayed. If you're curious, you can still check out my developer account here. Pada Januari 2021, saya berhenti mengembangkan aplikasi desktop karena tidak ada platform periklanan yang sederhana. Sebagai gantinya, saya beralih ke pengembangan aplikasi seluler dengan Flutter untuk membuat aplikasi Android. Saya berhasil mengembangkan sekitar 10 aplikasi dan mengunggahnya ke Google Play Store. Namun, pada Oktober 2024, saya harus menyembunyikan semua aplikasi tersebut karena kebijakan baru Google yang mengharuskan alamat pribadi ditampilkan. Jika kamu penasaran, kamu masih bisa melihat akun pengembang saya di sini.

In August 2022, I purchased a domain called redmerah.com and hosting services. I learned HTML, CSS, JavaScript, and Express for back-end development, and built an article web app. This marked my first foray into web app development. Since then, I've explored various web frameworks and libraries like React, Vue.js, Nuxt, Next.js, Svelte, SvelteKit, SolidJS, SolidStart, and Astro. Among these, I frequently use React (industry standard), Svelte + SvelteKit (great for multi-page apps), SolidJS (incredibly fast), and Astro (for static site generation). Pada Agustus 2022, saya membeli domain bernama redmerah.com dan layanan hosting. Saya mulai mempelajari HTML, CSS, JavaScript, dan Express untuk pengembangan back-end, serta membangun aplikasi web artikel. Ini menandai langkah pertama saya dalam pengembangan aplikasi web. Sejak saat itu, saya telah menjelajahi berbagai framework dan library web seperti React, Vue.js, Nuxt, Next.js, Svelte, SvelteKit, SolidJS, SolidStart, dan Astro. Di antara semuanya, saya sering menggunakan React (standar industri), Svelte + SvelteKit (hebat untuk aplikasi multi-halaman), SolidJS (super kencang), dan Astro (untuk generasi situs statis).

One key tool I rely on is Figma for prototyping any app I create, including layouts, icons, logos, and overall design. Salah satu alat utama yang saya andalkan adalah Figma untuk membuat prototipe aplikasi apa pun yang saya buat, termasuk tata letak, ikon, logo, dan desain keseluruhan.

While my primary focus is on web UI/UX, I'm eager to expand my skill set to include desktop and mobile app development, bringing cohesive and innovative designs across multiple platforms. Walaupun fokus utama saya adalah pada UI/UX web, saya sangat ingin memperluas keterampilan saya untuk mencakup pengembangan aplikasi desktop dan seluler, sehingga bisa menghadirkan desain yang kohesif dan inovatif di berbagai platform.
